# Shapes
Print the following shape to the screen:
```
    #
   ##
  ###
 ####
#####
```

The characters '#', '\n', and ' ' (space) may appear once and only once in the code.

A written description of the shape is:
- 4 spaces, one hash
- 3 spaces, two hashes
- 2 spaces, three hashes
- 1 space, 4 hashes
- 0 spaces, 5 hashes
